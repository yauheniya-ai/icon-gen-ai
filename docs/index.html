icon-gen-ai

AI-powered icon generator from Iconify, direct URLs, or local files.

Try it


pip install icon-gen-ai
icon-gen-ai generate -i input/pypi-icon.svg -o output/pypi-icon.svg --bg-color '(tan,cyan)' --size 128 --border-radius 64 
pip install icon-gen-ai[ai]
[ -f .env ] && echo ".env exists" || echo ".env doesn't exist, alternative: export OPENAI_API_KEY=sk-proj-..."
icon-gen-ai providers
icon-gen-ai search "GitHub logo and icons"
icon-gen-ai generate simple-icons:github -o output/github-icon.svg --bg-color '(#8b76e9,#ea2081)' --color black --size 128 --border-radius 64
